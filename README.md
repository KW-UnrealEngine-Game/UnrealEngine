# 🎮 IN TO THE DUNGEON

> **쉽고 함께 즐기는 ARPG**  
> 진입 장벽을 낮추고, 협동과 성장의 재미를 강화한 액션 RPG

---

## 📌 프로젝트 소개

기존 ARPG 게임들은 높은 난이도와 복잡한 시스템으로 인해  
초보 유저나 라이트 유저가 접근하기 어렵다는 한계가 있습니다.

본 프로젝트는 이러한 문제를 해결하기 위해  
**간단한 조작**, **명확한 성장**, **협동 플레이**에 집중한 ARPG 게임입니다.

- 복잡한 조작 없이도 전투의 재미를 느낄 수 있도록 설계
- 혼자서도, 여럿이서도 부담 없이 즐길 수 있는 콘텐츠 구성
- 반복 플레이를 통해 자연스럽게 실력이 성장하는 구조

---

## ❗ 기존 ARPG 게임의 한계

- 높은 진입 장벽으로 인한 신규 유저 이탈
- 복잡한 스킬·아이템 시스템으로 학습 부담 증가

➡️ **본 프로젝트는 ‘쉽고 함께 즐기는 ARPG’를 목표로 합니다.**

---

## 🎯 기획 방향

### ✔ 쉽고 함께 즐기는 게임
- 단순하지만 직관적인 전투 시스템
- 파티 플레이 시 자연스럽게 협동이 이루어지는 구조

### ✔ 진입 장벽을 낮춘 ARPG
- 필수 조작 최소화
- 플레이를 통해 자연스럽게 시스템을 익히도록 설계
